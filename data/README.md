# 📂 Data Directory

This directory is used to store datasets required for training and evaluation.

Due to GitHub's file size limitations, datasets are **not included** in this repository.

## Download

Download the dataset from the following source and place it in this directory:

- **Ransomware Dataset** — [Kaggle: Ransomware Detection using ML](https://www.kaggle.com/)

### Expected Files

```
data/
├── Ransomware.csv
└── new_dataset/
    ├── API_Functions.csv
    ├── DLLs_Imported.csv
    ├── PE_Header.csv
    └── PE_Section.csv
```

> ⚠️ These files are listed in `.gitignore` and will not be tracked by Git.
